# How It Works

## Smart Traffic Intersection Agent

The Smart Traffic Intersection agent monitors traffic intersection at real time by processesing real-time traffic data from Message Queuing Telemetry Transport (MQTT) streams and providing advanced analytics on, but not limited to, directional traffic density, Vision Language Model (VLM)-powered traffic scene, and traffic.

The agent supports sliding-window analysis, sustained traffic detection, and intelligent management of camera images to enhance traffic insights.
